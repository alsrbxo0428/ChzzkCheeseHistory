## 치지직 치즈 후원 내역
링크 : https://alsrbxo0428.github.io/ChzzkCheeseHistory/

개인적으로 스트리머별 후원 내역을 확인하고 싶어 제작했으며, Windows OS 및 PC에서의 사용을 전제로 제작하였습니다.\
